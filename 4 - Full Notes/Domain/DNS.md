# Conceito Geral
**Domain Name System** - É o livro de endereços da internet é ele que amarra o "www.dominio.com" ao seu endereço [[IP]].
O funcionamento do DNS é complexo, mas da para resumir em um sistema distribuído de servidores onde quando uma requisição a um domínio é feita, esse sistema se resolve por meio de varias conversas até achar o endereço IP a qual esse domínio corresponde.
Essa rede contém 4 tipos de servidores:
- **[[DNS Resolver]]**
- **[[Root Name Server]]**
- **[[Top-Level Domain Name Server]]**
- **[[Authority Name Server]]**

# Conceito Geral
Basicamente consiste em duplicar dados que já existem em outro lugar, os dados de cache são geralmente chamados de **dados quentes**. Essa duplicação é excelente, pois, os dados são armazenados em alguma ferramenta especifica para isso que é extremamente rápida de ser acessada, inclusive, existem hardwares que são específicos para isso, que são bem mais caros que os normais.
## Quando utilizar
Quando você tem dados que tem alta taxa de acessos, pouca alteração e precisam ser disponibilizados de forma rápida.
## Quando não utilizar
Nunca utilizar para ser o seu armazenamento principal de dados, pois além de caro, é um armazenamento volátil.
